# AI Lead Portfolio (GitHub Pages)

This is a **static** GitHub Pages site (no Jekyll required).  
Edit `assets/config.js` to set your name + links, then push to GitHub and enable Pages.

## Quick start (GitHub Pages)
1. Create a new repo (example: `ai-lead-portfolio`)
2. Copy these files into the repo root
3. In GitHub: **Settings → Pages**
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
4. Save — your site will be available at the Pages URL.

## Customize
- Update your info in `assets/config.js`
- Replace the placeholder resume: `assets/Chris_Tortorella_Resume.pdf`
- Update each project card in `projects.html` with your repo/demo links.
