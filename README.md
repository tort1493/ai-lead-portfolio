# AI Lead Portfolio (GitHub Pages)

This is a **static** GitHub Pages site with no Jekyll requirement.
Edit `assets/config.js` to set your name and links, then push to GitHub and enable Pages.

## Quick start
1. Create or use the `ai-lead-portfolio` repository.
2. Commit these files to the repo root.
3. In GitHub, open **Settings -> Pages**.
4. Set the source to `Deploy from a branch`.
5. Select the `main` branch and `/ (root)` folder.
6. Save and wait for the Pages deployment URL.

## Customize
- Update your info in `assets/config.js`
- Update the linked resume asset in `assets/`
- Update project cards in `projects.html` with repo, docs, and live-demo links
